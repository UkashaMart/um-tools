# Strong Password Kaise Banayein

Aajkal online security bahut zaroori hai. Weak password se aapka account hack ho sakta hai.

## Strong Password ke Features

- **Lambi:** Kam se kam 12-16 characters
- **Mix:** Uppercase, lowercase, numbers, symbols
- **Unique:** Har account ke liye alag password
- **Random:** Koi naam ya date mat use karo

## Common Mistakes

❌ `password123` — Bahut common
❌ `Ali1990` — Personal info
❌ `123456` — Bilkul simple

## Strong Password Examples

✅ `K8#mP2$vL9@nQ5`
✅ `Xy7!rB3&wN1*tF`
