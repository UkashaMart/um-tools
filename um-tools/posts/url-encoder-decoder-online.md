# URL Encoder Decoder Online Free

Jab aap URL mein special characters use karte hain to browser unhe encode karta hai. Hamara tool manually encode/decode karne mein help karta hai!

## URL Encoding Kya Hai?

URLs mein sirf specific characters allowed hain. Special characters ko `%XX` format mein encode kiya jata hai.

## Common Encoding Examples

| Original | Encoded |
|----------|---------|
| Space | %20 |
| @ | %40 |
| & | %26 |
| = | %3D |
| + | %2B |
| / | %2F |

## Example

Original URL:
`https://example.com/search?q=hello world&lang=ur`

Encoded:
`https://example.com/search?q=hello%20world&lang=ur`

## Kab Use Karein?

- API parameters mein special characters
- Query strings mein Urdu/Arabic text
- Form data submission
- Redirect URLs
