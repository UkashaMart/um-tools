# Percentage Calculator Online Free

Percentage calculations roz ki zindagi mein bahut common hain — discount, tax, marks, salary increase — sab mein percentage use hoti hai.

## Hamara Tool 3 Types Calculate Karta Hai

### Type 1: X ka Y% kya hoga?
Example: 5000 ka 18% = 900

**Use:** GST/Tax calculate karna, discount amount

### Type 2: A, B ka kitna % hai?
Example: 45 marks out of 60 = 75%

**Use:** Test marks percentage, business share

### Type 3: X se Y tak percentage change?
Example: 1000 se 1200 = 20% increase

**Use:** Salary increase, price change, profit/loss
