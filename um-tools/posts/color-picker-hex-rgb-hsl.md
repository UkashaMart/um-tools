## Color Formats

### HEX Format
6 digit hexadecimal code.
Example: `#D4A017` (Gold color)

### RGB Format
Red, Green, Blue values (0-255).
Example: `rgb(212, 160, 23)`

### HSL Format
Hue, Saturation, Lightness.
Example: `hsl(43°, 80%, 46%)`

## Color Tips for Designers

1. **Contrast ratio** 4.5:1 hona chahiye accessibility ke liye
2. **60-30-10 rule** — Primary, secondary, accent colors
3. **Brand consistency** — Har jagah same colors use karein
